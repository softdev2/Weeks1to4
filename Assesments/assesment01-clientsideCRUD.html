Assesment 01

Clientside CRUD

Title:
A Film webpage (CRUD)
Description:

Write a webpage that allows the user to perform CRUD operations on data of films

A film will have

  • id
  • name (String)
  • Date Published (date)

The website will allow the user to:

  1. view all the Films in a table
  2. Create a new Film
  3. delete a Film from the table
  4. update a Film in the table

The webpage does not need to store the data on a server. The page should look nice and be intutive to use
